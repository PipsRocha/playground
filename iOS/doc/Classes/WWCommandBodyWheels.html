Inherits from WWCommand : WWComponent : WWObject : NSObject
Declared in WWCommandBodyWheels.h

Overview

WWCommandBodyWheels objects instruct a WWRobot how to move by specifying the velocity for each wheel.

Valid wheel velocity values range from -100 to 100 cm/s. However, actual speed might vary based on surface conditions (e.g., carpet speed will be slower).
