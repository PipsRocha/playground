WWCommandEyeRing Class Reference

Inherits from WWCommand : WWComponent : WWObject : NSObject
Declared in WWCommandEyeRing.h

Overview

WWCommandEyeRing objects instruct a WWRobot how to display its eye pattern.

For ledBitmap, the mapping represents an analog clock, where index 0 represents the light at 12 o'clock, index 1 represents the light at 1 o'clock, index 2 represents the light at 2 o'clock, and so on.
