WWCommandSetSequence Class Reference

Inherits from WWEventDataSource : WWObject : NSObject
Declared in WWCommandSetSequence.h

Overview

WWCommandSetSequence represents a sequence of commands that can be executed serially. Each sequence has 1 or more WWCommandSetSequenceFrame objects, which describes a WWCommandSet to execute for the duration of that frame. This class is best for creating a robot animation sequence.
