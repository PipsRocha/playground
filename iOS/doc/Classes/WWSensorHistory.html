WWSensorHistory Class Reference

Inherits from WWObject : NSObject
Declared in WWSensorHistory.h

Overview

The WWSensorHistory object contains a set of WWSensor objects in reverse chronological order. New sensor data are coming in roughly every 30ms under normal conditions. However, depending on the signal strength of the robot, it may extend out to 40ms.

Returns the WWSensor object received seconds ago.

The returned WWSensor will be the closest WWSensor object received at the specified time or sooner. The WWSensor timestamp is used for this calculation. Nil is returned if no corresponding WWSensor is found.
